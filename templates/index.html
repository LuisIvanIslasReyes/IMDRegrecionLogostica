Modelo de Regresión Logística

Predicción de comportamiento de compra basado en edad y salario estimado

Nuestro modelo utiliza técnicas de machine learning para predecir la probabilidad de que un usuario realice una compra basándose en sus características demográficas.

Características del Modelo

Regresión Logística

Utiliza algoritmos de machine learning para clasificación binaria con alta precisión en las predicciones.

Métricas Completas

Evaluación exhaustiva del modelo con accuracy, precision, recall, F1-score y análisis de la curva ROC.

Predicciones en Tiempo Real

Ingresa edad y salario para obtener predicciones instantáneas sobre la probabilidad de compra.

Información del Dataset

Características del Dataset:
  • User ID: Identificador único
  • Género: Masculino/Femenino
  • Edad: Edad del usuario
  • Salario Estimado: Ingresos anuales
  • Purchased: Variable objetivo (0/1)
Objetivo del Modelo:

El modelo predice si un usuario comprará un producto basándose en su edad y salario estimado. Esto permite identificar clientes potenciales y optimizar estrategias de marketing.

Aplicaciones: Marketing dirigido, análisis de segmentación de clientes, y optimización de campañas publicitarias.
